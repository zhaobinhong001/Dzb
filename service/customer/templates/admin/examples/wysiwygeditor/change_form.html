Notes on WYSIWYG editors

Examples you see above are django-suit-redactor and django-suit-ckeditor packages. We tested many existing apps, but many of them were missing tests, had implementation or other flaws, so we decided to create our own. We are planning to improve these packages in future (support file upload, etc.)

Keep in mind Imperavi Readactor is not part of Django Suit package and is licensed under Creative Commons Attribution-NonCommercial 3.0 license.

Also you can integrate WYSIWYG editor using any other third party packages.
More documentation on wysiwyg editors integration is here

Django Suit wysiwyg packages:
  • django-suit-redactor - Imperavi Redactor. No file upload yet (Planned)
  • django-suit-ckeditor - CK Editor. No file upload yet (Planned)
Few third party packages we tested for Django Suit:
  • django-tinymce - TinyMCE editor. Tested and works great
  • django-ckeditor - CK Editor. Works great, but no support for CK Editor 4.x
  • django-cked - CK Editor. Quite new and unfortunately buggy. Supports CK Editor 4.x
  • django-redactorjs - Imperavi Redactor.
  • django-redactor - Imperavi Redactor. No Pypi package yet
  • More here: WYSIWYG Editors grid on DjangoPackages.com
